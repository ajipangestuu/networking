Enable DHCP on the Server:
○ Go to Services > DHCP.
○ Enable the DHCP service by toggling it to “On.”
○ Configure a DHCP pool for each subnet:

■ For VLAN 10 (MARKETING):
■ Pool Name: Sales
■ Default Gateway: 192.168.10.1
■ Start IP Address: 192.168.10.10
■ Subnet Mask: 255.255.255.0
■ Maximum Users: 20

■ For VLAN 20 (IT):
■ Pool Name: IT
■ Default Gateway: 192.168.20.1
■ Start IP Address: 192.168.20.10
■ Subnet Mask: 255.255.255.0
■ Maximum Users: 20

■ For VLAN 30 (HR):
■ Pool Name: HR
■ Default Gateway: 192.168.30.1
■ Start IP Address: 192.168.30.10
■ Subnet Mask: 255.255.255.0
■ Maximum Users: 20
